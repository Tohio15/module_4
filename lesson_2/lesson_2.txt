# HTML(Hypertext Markup Language) - язык гипертекстовой разметки, который используется при верстке веб-страниц.
# HTML-тег - команда, которая задает способ отображения информации в браузере.
# Теги бывают: парные и ординарные.

# CSS(Cascading Style Sheets) - код, который используется для стилизации веб-страницы.

# Фреймворк - набор библиотек, файлов, конфигураций и прочих инструментов и настроек,
 целью которых является решение определенного типа задач.

# Клиент-серверная архитектура - такая архитектура, в чьих рамках есть сервер,
 у которого есть необходимые данные (какая-либо информация), и клиенты,
  хотящие эти данные получить.

# GET-запрос - запрос,
 при помощи которого клиент запрашивает какую-либо информацию у сервера.
# POST-запрос - запрос,
 при помощи которого клиент отправляет на сервер какую-либо информацию.

# URI(Uniform Resource Identifier) - символьная строка,
 которая позволяет идентифицировать какой-либо ресурс: документ, изображение, файл и т.д.
